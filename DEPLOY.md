# Деплой Taplink-style

## Данные на диске

Задайте **`DATA_DIR`** (например `/app/data`) и примонтируйте **том** к этому пути, иначе профили и загрузки пропадут при пересоздании контейнера.

В каталоге появятся:

- `profiles/` — JSON профилей (в т.ч. `ownerTelegramId` для правки только владельцем)
- `avatars/`, `backgrounds/` — бинарные файлы
- `clicks/` — счётчики переходов по кнопкам

## Сборка фронта

Перед образом или выкладкой: `npm ci` и **`npm run build`** — в `dist/` должен попасть актуальный SPA.

## Безопасность правок

По умолчанию **PUT/POST/DELETE** к `/api/public/...` требуют заголовок **`X-Telegram-Init-Data`** (Mini App передаёт его сам).

- **`BOT_TOKEN`** обязателен для проверки подписи (и для бота).
- Только для отладки локально: **`ALLOW_INSECURE_EDIT=1`** (никогда в публичном проде без понимания рисков).

## Мониторинг

- Проверка живости: **`GET /health`**
- Настройте алерт при циклических рестартах контейнера и бэкап тома `DATA_DIR`.

## Резервные копии

Регулярно копируйте содержимое **`DATA_DIR`** (или снапшот тома).
